# Qur'oni Karim va Namoz Vaqtlari Ilovasi

Ushbu loyiha zamonaviy texnologiyalar yordamida yaratilgan, foydalanuvchilarga Qur'on o'qish va namoz vaqtlarini kuzatib borishda yordam beradigan premium veb-ilova.

## 🌟 Asosiy Imkoniyatlar

- **📖 To'liq Qur'on**: Barcha suralarning arabcha matni va tarjimalari.
- **✨ Mushaf Layout**: Oyatlar ketma-ket, xuddi haqiqiy Mushafdagidek chiroyli ko'rinishda joylashgan.
- **🎙️ Ovozli Qiroat Tekshiruvi**: Foydalanuvchi mikrofon orqali surani o'qiganda, dastur uni eshitadi, xatolarni qizil, to'g'ri o'qigan joylarini yashil rangda ko'rsatadi. Xato bo'lganda maxsus ovozli signal beradi.
- **🕌 Namoz Vaqtlari**: Foydalanuvchining joylashuviga (Geolocation) qarab, Aladhan API orqali eng aniq namoz vaqtlarini ko'rsatadi.
- **⏳ Qayta Hisoblagich**: Navbatdagi namoz vaqtigacha qancha vaqt qolganini real vaqt rejimida ko'rsatadi.
- **🎨 Premium Dizayn**: Tailwind CSS yordamida yaratilgan, ko'zni charchatmaydigan va chiroyli interfeys.

## 🚀 Texnologiyalar

- **React.js** - Frontend freymvork
- **Vite** - Tezkor yig'uvchi (Bundler)
- **Tailwind CSS** - Stil berish uchun
- **Lucide React** - Ikonkalar uchun
- **Web Speech API** - Ovozni aniqlash uchun
- **Axios** - API so'rovlari uchun

## 🛠️ O'rnatish va Ishga Tushirish

1. Loyihani yuklab oling.
2. Kerakli kutubxonalarni o'rnating:
   ```bash
   npm install
   ```
3. Loyihani ishga tushiring:
   ```bash
   npm run dev
   ```
